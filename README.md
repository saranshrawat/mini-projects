# Mini Projects

A monorepo containing various mini React projects for learning and experimentation.

## Projects

- **advice_creator**: A project for generating advice.
- **Pagination**: A pagination component example.
- **redux**: Redux examples including counter and todo list.
- **todo_list**: A simple todo list application.

## Getting Started

Each project has its own `package.json` and can be run independently.

To run a specific project:

```bash
cd <project-name>
npm install
npm start
```
