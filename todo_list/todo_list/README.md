# Todo List

A simple React application for managing a todo list with add and delete functionality.

## Features

- Add new todo items
- Delete items from the list
- Clean, minimal UI
- Real-time updates

## Technologies Used

- React (Functional Components, Hooks)
- CSS for styling

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm

### Installation

1. Navigate to the todo_list directory:
   ```bash
   cd todo_list/todo_list
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm start
   ```

4. Open [http://localhost:3000](http://localhost:3000) to view the app.

## Components

- `App.jsx` - Main component with state management
- `Todo_list.jsx` - Individual todo item component
- `Demo.jsx` - Additional demo component

## Available Scripts

- `npm start` - Runs the app in development mode
- `npm test` - Launches the test runner
- `npm run build` - Builds the app for production
- `npm run eject` - Ejects from Create React App (irreversible)
